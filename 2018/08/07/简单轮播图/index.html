发表于 2018-08-07 | 更新于 2019-04-05 | 分类于 前端小零件 | 评论数：

　　　　　　　　对于轮播图的实现有很多种方式，而我即将要写的这个是用的是偏移量来写的。

　　　　

　　　　一、写轮播图需要知道的几个知识点（是对于我写这个）：

　　　　1. 知道如何获取 html 里的元素，他有两种获取方法，第一个是用 getElementById
　　　 　　来获取元素，第二一个是一个新方法即 querySelector 来获取。

　　　　2.用 getElementsByTagName 来查询所有的图片和圆点。

　　　　3. 用 onclick 来触发事件．

　　　　4.设置轮播图自动滚动事件用到 setInterval 。

　　　　5..设置暂停轮播事件 clearInterval(timer) 。在这里包括：鼠标移出时用
　　　　 onmouseout 移入时用 onmouseover (让鼠标放在轮播图区域时滚动和离开时暂停)。

　　　　6. className 用于设置或返回元素的 class 属性。

　　　　以上的这些是写如下轮播图需要用到的，可能写的不详细，需要自己去查找上面每个的用法。
