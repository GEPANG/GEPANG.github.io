初识jQuery

发表于 2019-04-04 | 更新于 2019-04-05 | 分类于 javascript库 | 评论数：

　　　　现在的我在前端这条路上已经走了大约一年了，对于这方面的知识掌握的也不多，如今也学到了jQuery这
　　部分了，当然了对于前面的知识自己并没有去深究，只是了解皮毛吧，但对于学习jＱuery这部分应该是够的。
　　好了，话不多说，看下文讲解…

　　　　一、什么是jQuery
　　　　说简单点jQuery就是一个javascript的库。它是通过封装原生的javascript函数得来的，它的作者是John
　　 Resig，它创建与2006是一个开源的项目，随着越来越多的开发者加入，jQuery越来越强大了，至如今jQuery
　　已经集成了javascript，css，Dom和Ajax于一体的强大功能，如：它优化HTML文档操作、事件处理、动画设
　　计和Ajax交互，可以用最少的代码，完成更多复杂而困难的功能，也应此得到了许多开发的青睐。它还拥有便
　　捷的插件扩展机制和丰富的插件并且jQuery兼容各种主流浏览器，如IE 6.0+、FF 1.5+、Safari 2.0+、Opera
　　9.0+等等

　　　　二、学习jQuery的条件
　　　　由于jＱuery是javacript封装而来的，也应此没有javascript那么复杂，当然了学习起来也要容易一些，但
　　对于这网页编程来说，有些必备的基础是要有的，如果连它说什么alert，变量．．．都不知道那你就无法学习
　　了。

　　　　1. HTML，css这些网页必备的基础技术。
　　　　2. 了解javascript的基本语法知识。
　　　　3. php（这是后面要用的，我认为有前两个也是可以学习的）。

　　　　三、jQuery的功能和优势
　　　　1. 像css那样访问和操作DOM
　　　　2. 修改css控制页面外观
　　　　3. 简化javascript的代码操作
　　　　4. 事件处理更容易
　　　　5. 各种动画效果使用方便
　　　　6. Ajax技术更加完美
　　　　7. 自身基于jQuery还有大量的插件使用更为方便

　　　　四、其他的javascript库
　　　　目前除了jＱuery外还有5个库较为流行，他们分别是YUI,Prototype，Mootools,Dojo和ExJs。
　　　　1.YUI，是雅虎公司开发的一套完整的，扩展性良好的交互网页工具。
　　　　2.Prototype，是最早成型的javascript库之一。
　　　　3.Dojo,Dojo强大之处在于提供了其他库没有的功能，如：离线存储，图标组件等等。
　　　　4.Mootools,是轻量，简洁模块化的javascript的库。
　　　　5.ExJs,简称Ext,原本是对YUI的一个扩展，主要创建前端用户界面。（付费）
